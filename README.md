## Payment & Fintech APIs

- [M-Pesa Daraja API](https://developer.safaricom.co.ke/) - Safaricom's official API for M-Pesa integration — STK Push, B2C, C2B, and more.
- [Flutterwave](https://developer.flutterwave.com/) - Payment infrastructure for Africa — accept payments from cards, mobile money, bank transfers across 30+ countries.
- [Paystack](https://paystack.com/docs/) - Simple, modern payments API for Africa (acquired by Stripe) — powers businesses across Nigeria, Ghana, Kenya, and South Africa.
- [Africa's Talking](https://africastalking.com/) - APIs for SMS, USSD, voice, airtime, and payments across Africa.
- [Chipper Cash](https://chippercash.com/) - Cross-border payment platform for free P2P transfers across Africa.
- [IntaSend](https://intasend.com/) - Kenya-focused payment gateway with M-Pesa, card, and Bitcoin support.
- [Kopokopo](https://kopokopo.co.ke/) - Merchant payment solutions built on M-Pesa for Kenyan businesses.
- [Cellulant](https://www.cellulant.io/) - Pan-African payment platform connecting businesses to 35+ countries.
- [Mono](https://mono.co/) - Financial data API for connecting bank accounts across Africa (Plaid for Africa).
- [Stitch](https://stitch.money/) - Financial data and payment API for South African financial institutions.
- [Pawapay](https://www.pawapay.io/) - Mobile money API platform covering 25+ African markets.
- [Thepeer](https://thepeer.co/) - API for enabling direct transfers between African fintech businesses.
- [OPay](https://www.opayweb.com/) - Mobile payment platform and fintech super-app operating across Africa.

## Open Source from Africa

- [Appwrite](https://github.com/appwrite/appwrite) - Open-source backend-as-a-service (co-founded by Eldad Fux, with significant African contributor community).
- [Chatwoot](https://github.com/chatwoot/chatwoot) - Open-source customer engagement platform with strong African community adoption.
- [Paystack CLI](https://github.com/PaystackOSS/paystack-cli) - Official CLI tool for interacting with the Paystack API.
- [africastalking-node](https://github.com/AfricasTalkingLtd/africastalking-node.js) - Node.js SDK for Africa's Talking APIs.
- [flutter_paystack](https://github.com/wilburt/flutter_paystack) - Flutter plugin for Paystack payment integration.
- [daraja-go](https://github.com/jwambugu/mpesa-golang-sdk) - Go SDK for M-Pesa Daraja API integration.
- [ussd_engine](https://github.com/habbes/ussd-go) - Go library for building USSD applications.
- [Ocrarium](https://github.com/nicholaskajoh/ocrarium) - Computer vision project for Nigerian license plate recognition.

- [FairMoney](https://fairmoney.io/) - Mobile banking and microlending in Nigeria and India.

### Healthtech

- [mPharma](https://mpharma.com/) - Managing prescription drug inventory and financing for pharmacies across Africa.
- [54Gene](https://54gene.com/) - African genomics research and biobank company.
- [Helium Health](https://heliumhealth.com/) - Electronic health records and hospital management for Africa.
- [Flutterwave Health](https://www.reliance-hmo.com/) - RelianceHMO — health insurance tech platform in Nigeria.
- [Ilara Health](https://ilarahealth.com/) - Making diagnostics accessible at point-of-care in East Africa.

### Agritech

- [Twiga Foods](https://twiga.com/) - B2B food distribution platform connecting farmers to vendors in Kenya.
- [Apollo Agriculture](https://apolloagriculture.com/) - AI-powered farming packages (seeds, fertilizer, insurance, credit) for smallholder farmers.
- [Farmcrowdy](https://www.farmcrowdy.com/) - Digital agriculture platform connecting farmers with sponsors in Nigeria.
- [Aerobotics](https://www.aerobotics.com/) - AI-powered drone analytics for crop health and tree counting in South Africa.
- [iProcure](https://iprocure.co/) - Agricultural supply chain digitization in East Africa.

### Edtech

- [Eneza Education](https://enezaeducation.com/) - Mobile learning platform delivering education via SMS and app across Africa.
- [Moringa School](https://moringaschool.com/) - Tech education and bootcamps in Nairobi and remote.
- [AltSchool Africa](https://www.altschoolafrica.com/) - Online school for software engineering in Africa.

### Logistics & Mobility

- [Kobo360](https://www.kobo360.com/) - Uber for trucks — logistics marketplace connecting shippers and carriers.
- [Lori Systems](https://www.lorisystems.com/) - Technology-enabled logistics connecting cargo owners with transporters.
- [Gokada](https://www.gokada.ng/) - Last-mile delivery and ride-hailing in Lagos.
- [SafeBoda](https://safeboda.com/) - Motorcycle ride-hailing and delivery in Uganda and Kenya.
- [Sendy](https://www.sendy.co.ke/) - On-demand delivery platform for businesses in East Africa.

### E-commerce

- [Jumia](https://www.jumia.com/) - Africa's leading e-commerce platform (NYSE: JMIA).
- [Takealot](https://www.takealot.com/) - South Africa's largest online retailer.

- [TradeDepot](https://tradedepot.co/) - B2B e-commerce and distribution platform for emerging markets.
